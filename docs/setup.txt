create a react app (using vite : npm create vite@latest)
select your framework and setups

This will create a react js app

next : install tailwind css 
    ==> see https://tailwindcss.com/docs/installation/using-vite

run the app     
    ==> npm run dev

you can use tailwind css in the app now

(keep the app running)

next : install tanstack router
    ==> see https://tanstack.com/router/latest/docs/framework/react/quick-start
        npm install @tanstack/react-router
        npm install -D @tanstack/router-plugin @tanstack/react-router-devtools
add tanstackrouter in the vite plugin
    export default defineConfig({
        plugins: [
            tailwindcss(),
            tanstackRouter(),
            react()
        ],
    })

create a routes folder inside src folder
add a __root.tsx file there
    ==> you will see that the file has content automatically

go to App.tsx file
-modify it as:-
    import {RouterProvider, createRouter} from '@tanstack/react-router';
    import {routeTree} from "./routeTree.gen.ts";
    import "./App.css";

    const router = createRouter({ routeTree });

    function App() {
        return (
            <RouterProvider router={router} />
        );
    }

    export default App;

go to __root.tsx file
you will see
function RootComponent() {
  return (
    <React.Fragment>
      <div>Hello "__root"!</div>
      <Outlet />
    </React.Fragment>
  )
}

Outlet => route page (/ or /about or /contact)
<div>Hello "__root"!</div> => this will be visible on every page

create index.tsx page inside the routes folder ==> this will be / page
create about.tsx or about/index.tsx inside the routes folder ==> this will be /about page

React app with tailwind css and tanstack router is setup

next: add a header

create src/components/header/Header.tsx page

modify your RootComponent to:
function RootComponent() {
  return (
    <React.Fragment>
      <div className="">
        <Header />
        <Outlet />
      </div>
    </React.Fragment>
  );
}
